# Illumina Backdrops — Link in Bio

Custom link-in-bio page for [Illumina Backdrops](https://illuminabackdrops.etsy.com).

Live at: **https://illuminabackdrops.com**

## Tech

Static HTML/CSS/JS. No framework, no build step. Hosted on GitHub Pages with custom domain.

## Updating

Edit `index.html` directly. The featured hero section and link list are marked with HTML comments for easy seasonal updates.
